Help

Posts are written in Markdown, which is like a simpler form of HTML: 
Bold
**text**

Italic
_text_

Image
![Caption text](https://image/url.jpg)

Full-bleed image
![Caption text](https://image/url.jpg){.full-width}

Half-column image
![Caption text](https://image/url.jpg){.half-width}

Link
[Link text](https://url.com)

Section header
## Section title

Lists
- a
- b
- c

Footnotes
This is some text[^1].

[^1]: This is a footnote

Horizontal rule
---

And any HTML is valid too

Other tips:

  • Add gallery-image as a class to an a tag linking to an image to show it in a lightbox on click. For example: <a href="/uploads/foo.png" class="gallery-image"><img src="/uploads/foo.png"></a>
